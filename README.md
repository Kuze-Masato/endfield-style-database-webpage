# 终末地风格互动式百科图鉴
一款受《明日方舟：终末地》启发的互动式网页百科图鉴，支持用户可视化编辑、自定义内容条目，纯前端静态运行，无需后端依赖。

## ✨ 项目简介
本项目是一款基于 HTML、JavaScript 开发的静态网页百科工具，核心灵感来源于《明日方舟：终末地》的物品与资料展示风格，主打“轻量、易用、可自定义”。用户可通过专属后台编辑器，自由添加、修改物品、大类、小类等核心数据，所有内容实时存储在本地数据文件中，无需部署后端服务，打开网页即可使用，适合用于个人收藏、小型社群资料整理、游戏图鉴展示等场景。

## 🎯 核心功能
- 🔧 可视化编辑后台：通过 editor.html 页面，直观管理物品、大类、小类所有信息，操作简单，无需代码基础
- 📊 静态数据驱动：所有内容存储在 data.js 文件中，修改后实时生效，支持一键导出备份，方便分享与迁移
- 📚 三级分类导航：支持“大类-小类-物品”三级结构，例如“武器防具 → 武器/防具 → 具体物品”，导航清晰
- 🔍 搜索与展示：前端主页支持物品搜索，搭配图标化分类入口，快速定位目标内容
- 🚫 无后端依赖：纯前端开发，无需安装数据库、部署服务器，任意现代浏览器均可直接运行
- 📱 适配优化：兼容电脑端主流浏览器，界面布局简洁，操作流程流畅，支持数据实时预览

## 🚀 快速开始
### 环境要求
- 浏览器：Chrome、Edge、Firefox、Safari 等现代浏览器
- 无需安装 Node.js、Python 等额外环境，无需配置任何依赖

### 运行步骤
1.  克隆或下载项目到本地
    ```bash
    # 克隆仓库（若使用GitHub/GitLab）
    git clone https://github.com/jiushizhengren/endfield-encyclopedia.git
    
    # 若未使用版本控制，直接下载项目压缩包，解压后进入项目根目录即可
    ```
2.  打开网页
    - 打开项目根目录，找到 `index.html` 文件
    - 双击 `index.html`，或右键选择“用浏览器打开”，即可进入百科图鉴主页，查看现有内容
3.  编辑数据
    - 打开项目根目录的 `editor.html` 文件（同样用浏览器打开）
    - 进入编辑后台，可添加、修改、删除物品、大类、小类信息
    - 点击直连JS,连接后保存文件js文件也能同时修改
    - 编辑完成后，点击保存按钮，刷新 `index.html` 即可看到修改效果

## 📝 使用说明
### 1. 后台编辑操作（核心）
#### 1.1 物品管理
- 打开 `editor.html`，切换到“物品”标签页
- 点击“添加物品”，填写物品信息：ID（唯一，不可重复）、物品名称、稀有度、所属大类、所属小类、物品描述、图标路径（可选）
- 已添加的物品会显示在列表中，可点击“编辑”修改信息，点击“删除”移除物品
#### 1.2 分类管理
- 大类管理：切换到“大类”标签页，添加/编辑/删除一级分类（如“教学记录”“武器图鉴”“资源素材”）
- 小类管理：切换到“小类”标签页，为每个大类添加二级小类（如“武器防具 → 武器/防具”“资源素材 → 基础资源”）
#### 1.3 数据保存与预览
- 编辑过程中，点击“授权直连 JS”可实时预览修改效果，无需刷新页面
- 所有修改完成后，务必点击“导出/备份”，将生成的 `data.js` 文件替换项目根目录的原有文件，否则修改不会保存

### 2. 前端主页使用
- 打开 `index.html`，主页会显示所有大类的图标入口，点击任意大类，可进入该大类下的小类列表
- 点击小类，可查看该小类下的所有物品，点击物品可查看详细描述
- 页面搜索框可输入物品名称，实时筛选匹配的物品，快速查找目标内容
